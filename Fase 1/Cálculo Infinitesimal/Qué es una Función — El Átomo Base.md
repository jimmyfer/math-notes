# ⚙️ ¿Qué es una Función? — El Átomo Base

Una función $f(x)$ es una regla de correspondencia o una "máquina lógica":
* **$x$ (Input):** El valor independiente que introduces.
* **$f(x)$ o $y$ (Output):** El resultado que la máquina escupe.

## 🧠 Simplificación Algebraica:
Las funciones que parecen complejas a veces esconden formas simples. 
$$\text{Si } f(x) = \frac{x^2 - 1}{x - 1} \implies f(x) = \frac{(x-1)(x+1)}{x-1} \implies f(x) = x + 1$$

* **Geometría:** Al reducirse a $x + 1$, el gráfico resultante en GeoGebra es una línea recta infinita con una pendiente de $1$ (inclinación de $45^\circ$).
